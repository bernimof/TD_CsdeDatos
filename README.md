# Portafolio de ejercicios y casos (Talento Digital 2025–2026)
Repositorio que reúne análisis de casos y actividades desarrolladas durante el bootcamp de Ciencia de Datos.
El objetivo es evidenciar habilidades de análisis, limpieza y manipulación de datos.


## Herramientas utilizadas
- Python
- NumPy
- Pandas
- Jupyter Notebook (VSCode)
- Google Colab
